# Living at the Edge

A scroll-driven teaching page on human-elephant conflict, data literacy, and environmental justice at Kui Buri National Park, Thailand. Built for teaching School for Field Studies students.

Open `index.html` in any browser, or view the published page once GitHub Pages is enabled (Settings then Pages, deploy from the main branch, root).

The page runs on the project's real data (Department of National Parks encounter archive, 2020 to 2023; household survey, Ruam Thai, 2024), embedded directly in the file, so it works offline with no dependencies. Banner images are illustrative; every map, chart, and figure is drawn from the project data.

Chapter 3 fieldwork, Tyler Nuckols.
